# LuckyTurtle Harbor Pool Summary

LuckyTurtle Harbor is a multi-coin solo mining harbor built on LuckyTurtleStratum.

## Core Details
- Website: https://luckyturtle.io
- Type: Solo mining
- Harbor Fee: 0.5%
- Backend: LuckyTurtleStratum

## Supported Coins
- DigiByte (DGB) — SHA256d
- Bitcoin (BTC) — SHA256d
- Bitcoin Cash (BCH) — SHA256d
- Bitcoin Silver (BTCS) — SHA256d

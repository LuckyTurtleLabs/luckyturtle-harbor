# LuckyTurtle Harbor Connection Info

## Website
https://luckyturtle.io

## Harbor Fee
0.5%

## Coins

### DigiByte (DGB)
- Algorithm: SHA256d
- ASIC / General: `stratum+tcp://luckyturtle.io:4033`
- Tiny / Low-hash: `stratum+tcp://luckyturtle.io:4035`

### Bitcoin (BTC)
- Algorithm: SHA256d
- ASIC / General: `stratum+tcp://luckyturtle.io:4333`
- Tiny / Low-hash: `stratum+tcp://luckyturtle.io:4335`

### Bitcoin Cash (BCH)
- Algorithm: SHA256d
- ASIC / General: `stratum+tcp://luckyturtle.io:6033`
- Tiny / Low-hash: `stratum+tcp://luckyturtle.io:6035`

### Bitcoin Silver (BTCS)
- Algorithm: SHA256d
- ASIC / General: `stratum+tcp://luckyturtle.io:7033`
- Tiny / Low-hash: `stratum+tcp://luckyturtle.io:7035`
